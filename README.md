# 高质量数据集加工平台 (HDPP)

> 面向出版社的数据集加工管理后台 · 江苏凤凰科技出版社
>
> 基于《高质量数据集加工平台 PRD v1.0》开发，覆盖从「中台资源接收 → 数据清洗 → 标注加工 → 质检审核 → 成果回流」的全链路数据流转，内置 AI 预标注能力，打通知识数据中台上下游。

## ✨ 核心特性

### 🔄 全链路数据流转
- **中台接收队列**：知识数据中台结构化成果自动推送至待加工队列，支持认领、创建任务
- **数据清洗流水线**：7 种清洗规则（精确去重 / 近似去重 MinHash / 格式规范化 / 噪声过滤 / 质量初筛 / 隐私脱敏 / 语言过滤）
- **多范式标注工作台**：按任务类型动态渲染四套 UI
  - 📝 **文本分类** — 标签选择 + AI 预测 + 一键采纳
  - 🎯 **NER 序列标注** — 划词即标 + 实体高亮 + 角标 + 快捷键
  - 💬 **问答对生成（SFT）** — 原文 + 问题 / 答案双栏 + AI 生成建议
  - ⚖️ **偏好标注（RLHF）** — A / B 双栏择优
- **质检 → 仲裁 → 审核 → 发布 → 回流** 完整状态机联动

### 🤖 AI 预标注真功能
- 项目级配置：启用开关 / 预标注模型（中台模型网关）/ 置信度阈值 / 覆盖策略
- 工作台顶部 AI 状态栏：覆盖条目数、采纳率、模型来源
- 分类场景：Top-3 标签置信度展示，高置信度一键采纳，低置信度橙色警告
- NER 场景：AI 预标注实体以虚线高亮显示，角标标注 `AI·实体类型·94%`，可确认 / 修改 / 删除
- 项目统计：采纳率、修改率、提效百分比

### 🛡️ 版权约束联动
- 版权字段从中台接收时自动继承至 HDPP 数据集
- 禁训资源：导出格式列表中 HuggingFace 不可选
- 禁交付资源：导出操作整体被拦截
- 禁 AI 应用资源：回流至中台被拦截

### 🎨 精致 UI 设计
- 字体系统：Inter + Noto Sans SC（思源黑体）+ JetBrains Mono
- 配色：靛蓝渐变体系（brand #6366f1 + accent 紫色系）
- 玻璃态登录页、卡片悬浮微交互、modal 入场动画
- 亮 / 暗主题切换
- 响应式布局

## 🎯 NER 划词即标交互

NER 工作台是本平台的核心交互创新：

1. **划选原文** → 弹出就近标签浮窗
2. **选择标签**（点击或快捷键 1-5）→ 文字变色高亮 + 角标
3. **AI 预标注** → 虚线高亮显示，角标 `AI·机构·94%`
4. **实体操作** → 点击实体可修改类型 / 删除
5. **底部实体列表** → 实时更新，支持撤销 / 全部清除
6. **提交** → 实体数据写入 store，进度 +1，自动跳下一条

| 状态 | 边框 | 背景透明度 | 角标 |
|------|------|-----------|------|
| AI 预标注（待确认） | 虚线 dashed | 12% | `AI·机构·94%` |
| 人工确认采纳 | 实线 solid | 22% | `机构` |
| 人工新增 | 实线 solid | 22% | `机构` |
